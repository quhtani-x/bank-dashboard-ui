# Bank Dashboard UI

A good-looking banking dashboard, front-end only (all fake data). Built to
practice layout and make something that actually looks like a real product.
Has a sidebar, a credit card visual, summary stats, a spending chart and a
transactions list.

## features

- sidebar nav + topbar
- gradient credit card mockup
- animated balance counter
- weekly spending bar chart (Chart.js via CDN)
- recent transactions list

## run

just open `index.html` in a browser. needs internet for the Chart.js CDN.
